La création d'un site web pour les non développeurs (gratuitement et rapidement) [Maj. 2011-04-05]

April 5 2011, 6:55 AM  by Karim Cassam Chenaï

Pour créer un site web en quelques minutes avec 0€ et sans compétence technique, vous pouvez choisir l'une de ces plateformes, qui s'adressent notamment aux non développeurs :

  • Orienté site web “statique”:

    • http://weebly.com/
    • http://orbs.com/
    • http://sites.google.com/
    • http://roxer.com/

  • Orienté blog :

    • http://posterous.com/
    • http://tumblr.com/
    • http://blogger.com/
    • http://twitter.com/

  • Maj : Flavors.me est probablement la meilleur solution. En tout cas, c'est celle que j'ai utilisé pour mon site.

Conclusion : Ne dérangez les agences web que pour des sites à valeur ajoutée et que vous ne pouvez pas faire vous-mêmes :)

Vos retours d'expériences dans les commentaires de ce post sont les bienvenus.
