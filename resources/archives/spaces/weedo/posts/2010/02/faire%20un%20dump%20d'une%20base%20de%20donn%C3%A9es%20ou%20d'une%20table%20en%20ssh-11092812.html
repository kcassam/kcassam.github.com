faire un dump d'une base de données ou d'une table en ssh

February 4 2010, 12:56 AM  by Karim Cassam Chenaï

db complète :
mysqldump -u root -p dbname > backup.db.sql

une table :
mysqldump -u USER -p DATABASE --tables TABLE > TARGET.sql
