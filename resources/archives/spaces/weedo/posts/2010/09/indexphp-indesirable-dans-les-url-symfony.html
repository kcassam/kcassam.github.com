Question/Réponse : index.php indésirable dans les url symfony

September 23 2010, 8:52 AM  by Karim Cassam Chenaï

Question

J’ai un petit souci avec certaines des urls qui sont générées. Pour une url notée dans routing.yml:

outilssimulation:
  url:   /rachat-de-credit/simulation-rachat-de-credit/
  param: { module: outilssimulation, action: index}

Elle affiche sur la page web :

index.php/rachat-de-credit/simulation-rachat-de-credit/

L’idéal étant :

rachat-de-credit/simulation-rachat-de-credit/

Aurais tu une idée de ce qui génèrerais cette anomalie ?

Réponse

La réponse est quelque part là dedans : http://www.google.fr/search?q=symfony+no_script_name
