easy install of hudson on MAC OS X

December 17 2010, 8:20 AM  by Stevan Gaillard

 

Here are the steps to use hudson on mac, without installing a server like TomCat or GlassFish :

download the latest hudson build : http://hudson-ci.org/latest/hudson.war

create a shell file :

#!/bin/sh
nohup java -jar /Users/stevan/downloads/hudson.war

or any directory u want to put.

Go to system preferences, accounts, then open the "Login Items" panel (in french "Ouverture")

Then add the shell file in the list and, at your next login, hudson will start automatically.

 

the only annoying thing is the terminal window wich have to stay open, if you close it, hudson will stop...

 

That's all, your done with the installation of hudson, now you can start using it by accessing http://localhost:8080/ (by default)
