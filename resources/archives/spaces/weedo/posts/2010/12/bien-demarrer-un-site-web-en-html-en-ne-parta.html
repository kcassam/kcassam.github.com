Bien démarrer un site web en html en (ne) partant (pas) de zéro

December 27 2010, 3:20 PM  by Karim Cassam Chenaï

On sait démarrer un projet web côté serveur : LAMP stack + Symfony et/ou Wordpress. Côté client/template, les choses sont un peu moins claires.

Voici comment on peut procéder :

  • Couche “Template” : Utiliser http://fr.html5boilerplate.com/. HTML5 Boilerplate est un template HTML/CSS/JS de base (“starter”) qui contient un nombre de fonctionnalités hallucinant.

  • Couche “Layout” Utiliser http://www.yaml.de/en/home.html pour positionner les éléments : colonnes, teaser, footer, etc. La version actuelle est la 3.3. Il existe un builder pour créer visuellement le positionnement, mais ce builder utilise la version 3.0.6 de YAML. La méthode manuelle est peut-être mieux. En tout cas, YAML est abondemment documenté.

  • Qu'en pensez-vous ? Comment démarrer un projet html ? Ça intéresserait quelqu'un un package intégré comprenant MAMP ou LAMP+Symfony+Wordress+HTML5B+YAML ?
