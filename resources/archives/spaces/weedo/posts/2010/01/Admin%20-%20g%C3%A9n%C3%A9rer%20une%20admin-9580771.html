Admin - générer une admin

January 7 2010, 5:55 AM  by Ballut Vincent
Symfony peut créer une interface administration servant à gérer les enregistrements en base de données via quelques lignes de commandes simples. Avant tout, il nous faut déjà avoir configuré une base de donnée et une table (voir les billets sur la config BDD). En résumé, configurer une table pour Symfony revient à modifier le fichier schema.yml dans config. Symfony va de baser sur ce fichier pour générer l'admin.

https://gist.github.com/271233

https://gist.github.com/271234

Mon module s'appelle donc "questions"

Ce n'est pas dans la doc, mais pour que tout fonctionne, il faut également saisir les lignes de commande suivantes, qui servent à créer certaines classes indispensables :

https://gist.github.com/271235

Et voilà, on a maintenant une interface admin basique permettant de créer, supprimer, modifier et filtrer nos enregistrements. :)
