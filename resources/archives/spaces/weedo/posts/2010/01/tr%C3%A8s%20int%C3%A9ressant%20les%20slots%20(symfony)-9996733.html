très intéressant les slots (symfony)

January 15 2010, 10:11 AM  by Karim Cassam Chenaï
Lire la partie sur les slots :
http://www.symfony-project.org/book/1_2/07-Inside-the-View-Layer

Ça permet de modifier le layout à partir d'un template. Très puissant.

Exemple d'utilisation dans RCD, pour avoir le rss différent pour la partie cms :
Dans /sf/apps/rcd/modules/wp/templates/indexSuccess.php :
<?php slot('rss') ?> <!-- custom rss code for the current template--> <!-- wp : a changer--> <link rel="alternate" type="application/rss+xml" href="/wp/feed/" /><?php end_slot() ?>

Dans /sf/apps/rcd/templates/layout.php :
<?php if (has_slot('rss')): ?> <?php include_slot('rss') ?> <?php else: ?> <!-- default rss code --> <link rel="alternate" type="application/rss+xml" href="/flux" /> <?php endif; ?>

c'est tout !
