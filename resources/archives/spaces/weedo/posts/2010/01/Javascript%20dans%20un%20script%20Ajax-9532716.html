Javascript dans un script Ajax

January 6 2010, 5:44 AM  by Ballut Vincent
Je continue sur ma série Ajax...

Par défaut, les scripts Javascripts présents dans un template généré
par une requête en Ajax sont ignorés (pour des raisons de sécurité).
Pour les activer, il suffit d'ajouter l'option script = true :

https://gist.github.com/270280

Attention, si le Javascript est activé, il sera bien exécuté, mais il
n'apparaîtra pas dans Firebug ou autre... Nul mais normal d'après la
doc : "Even if you enable script execution for the remote response,
you won't actually see the scripts in the remote code, if you use a
tool to check the generated code. The scripts will execute but will
not appear in the code. Although peculiar, this behavior is perfectly
normal."
