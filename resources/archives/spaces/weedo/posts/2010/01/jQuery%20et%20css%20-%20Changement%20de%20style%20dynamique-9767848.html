jQuery et css - Changement de style dynamique

January 11 2010, 6:08 AM  by Karim Cassam Chenaï
Changement de style dynamique :
Des API jQuery permettent de modifier les propriétés css : http://docs.jquery.com/CSS

notamment : css( name, value )

Simple et pratique, mais il est beaucoup plus cohérent d'utiliser les API de changement de classe et de définir les propriétés css dans les fichiers css

http://docs.jquery.com/Attributes

Voir notamment : removeClass( class ) et addClass( class )

Ainsi, le style n'est à changer qu'à un seul endroit (fichier css) au lieu de 2 (fichier css et fichier js)
