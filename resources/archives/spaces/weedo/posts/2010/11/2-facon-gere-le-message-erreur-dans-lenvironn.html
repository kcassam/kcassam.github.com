2 façon géré le message erreur dans l'environnement xAMP

November 3 2010, 6:40 AM  by email
Statiquement:

Configurer les paramètres dans fichier php.ini,plusieurs éléments dans la partie error avec d'explication, n'oublie pas de relance le serveur une fois la modification est fait.

Dynamiquement:

Utiliser ini_set() pour géré l'affichage d'erreur dans un script particulier qu'on ne veut pas prendre trop de risque, par exemple: 

ini_set('display_errors', 1);

permet d'activé l'affichage d'erreur pendant exécution du script dernier.

PS: le fichier log est aussi très utile pour le debug, par exemple,on peut utiliser error_reporting() en le fournissant des paramètres pour voir un affichage dans le fichier php_error.log situe :xAMP/log/
