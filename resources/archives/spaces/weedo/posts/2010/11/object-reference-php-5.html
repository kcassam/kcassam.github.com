Object reference PHP 5

November 17 2010, 12:53 AM  by fix
Depuis PHP 5, tous les objets sont passes par reference.

Exemple:
$instance = new Instance();
$reference = $instance;     => $reference est une reference sur $instance, quand une des attributs de $instance sera modifie, $reference prendra en compte les modifications puisque il pointe sur $instance.

Pour dupliquer un objet, il faut utiliser la fonction clone($objet).

Le destructeur de l'objet sera automatiquement appele des que toutes les references sur cet objet sont supprimes ou quand l'objet est supprime explicitement (notamment via unset()).

lien explicatif: http://php.net/manual/en/language.oop5.decon.php
