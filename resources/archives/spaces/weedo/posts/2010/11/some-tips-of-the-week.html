How to log debug messages in firebug instead of using invasive alerts

Firebug

Use javascript & firebug like this :

console.log("hello world")

instead of this :

alert("goodbye world")
  • example : http://benalman.com/code/projects/javascript-debug/examples/debug/
  • tuto : http://michaelsync.net/2007/09/09/firebug-tutorial-logging-profiling-and-comm...
  • ref : http://getfirebug.com/logging

Php

With php, you echo a javascript console.log or, you can use as well firephp which is a php class to send console.log via response headers and will not interfere with the content on your page.
http://www.firephp.org/

Symfony built-in debugger

End Of Line with PHP

Please use the constant PHP_EOL instead of “\n”.
You can also check other useful PHP Predefined Constants : http://php.net/manual/en/reserved.constants.php

include a file

Php

you can include files with include() or require()
You can specify the path to the dirname( __FILE__ ) :

require( dirname( __FILE__ ) . '/../../subdir/somefile.php' );

http://stackoverflow.com/questions/786542/including-files-without-having-to-s... Do not use :

require(../../subdir/somefile.php' );

Because if your file is an included file as well from another folder, this will not work (the path will be resolved from the parent).
