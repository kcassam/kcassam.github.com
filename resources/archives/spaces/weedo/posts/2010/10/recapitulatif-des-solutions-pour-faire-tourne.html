Récapitulatif des solutions pour faire tourner des applis Windows sur Mac

October 5 2010, 6:02 AM  by Karim Cassam Chenaï

Ces solutions permettent de faire tourner des applications Windows sur Mac sans avoir à mettre les mains dans le cambouis :

  1. Crossover : http://www.codeweavers.com/products/cxmac/ –> Cette solution n'exige pas l'installation de Windows. Pas compatible avec tous les logiciels (liste des applications compatibles : http://www.codeweavers.com/compatibility/)
  2. BootCamp : http://www.apple.com/fr/support/bootcamp/ –> Cette solution n'exige pas l'installation d'une application tiers mais exige une installation Windows. Ne permet pas de lancer Windows et Mac OS X simultanément.
  3. Parallels Desktop : http://www.parallels.com/fr/products/desktop/ –> Cette solution exige l'installation d'une application tiers et de Windows. Permet de lancer Windows et Mac OS X simultanément.
  4. VMware Fusion : http://www.vmware.com/fr/products/fusion/ –> challenger de Parallels Desktop.
  5. Via un client VNC : Cette solution exige de disposer d'un PC Windows allumé et accessible sur le réseau, auquel on se connecte via un client VNC (par exemple Chicken of the VNC).

Vous en connaissez d'autres ? Quelle est votre solution préférée ? Pour quel usage ?
