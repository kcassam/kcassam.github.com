Var_dump() en JS

July 19 2010, 3:15 AM  by mysterty

Une fonction pratique que j'ai trouvé sur le net :

[code:javascript]function dump(obj) {
    var out = '';
    for (var i in obj) {
        out += i + ": " + obj[i] + "\n";
    }

    alert(out);

    // or, if you wanted to avoid alerts...

    var pre = document.createElement('pre');
    pre.innerHTML = out;
    document.body.appendChild(pre)
}[code]
