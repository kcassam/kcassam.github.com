protéger les dossiers sensibles

December 23 2009, 1:27 AM  by Karim Cassam Chenaï
Il est possible de protéger les dossiers pour lesquels vous souhaitez interdire un accès via http.

Il suffit de créer un fichier .htaccess dans le dossier et d'y mentionner l'instruction suivante :#====
Deny from all
#====

C'est utilisé cette technique pour le dossier trace par exemple.
Il est encore mieux de mettre ces dossiers dans le dossier sf, qui n'est pas accessible depuis l'extérieur.Il est aussi possible d'utiliser les 2 techniques.

Pour des besoins plus fins, il existe des options plus sophistiquées.
