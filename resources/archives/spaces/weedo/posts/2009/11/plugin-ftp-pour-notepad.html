Plugin FTP pour Notepad++

November 12 2009, 3:37 AM  by Ballut Vincent
Petit plugin utile pour éditer des fichiers distants via FTP :

http://sourceforge.net/projects/npp-plugins/files/FTP_synchronize/FTP_synchronize_0_9_6_1_dll.zip

Pour installer le plugin : télécharger via le lien ci-dessus, placez le fichier dll dans le dossier plugin du répertoire d'installation de Notepad++, et relancez Notepad.

Si vous êtes sous Windows Vista, petite manip en plus pour que ça marche : une fois le plugin installé, cliquez sur l'icône Préférences -> Général, et modifiez le répertoire du cache (5è option). Il faut que ce répertoire soit en dehors de Program Files, sinon Notepad n'arrivera pas à y enregistrer les fichiers cache.
