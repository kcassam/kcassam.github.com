jQuery et les évenements (onchange, onclic, onmouseover)

November 29 2009, 3:29 PM  by Karim Cassam Chenaï

jQuery propose une solution pour éviter de mélanger javascript et html : 

Au lieu de faire un 

<?php echo $form['pret_mensualite1']->render(array('class' => 'money','onchange'=>'FormatMyNombre(this);')); ?> €

on peut laisser l'input tranquille et écrire, dans la partie javascript : 

<script language="Javascript" type="text/javascript">
    //<![CDATA[
 jQuery(document).ready(function(){

        jQuery("#id_input").change(function () {
      FormatMyNombre(jQuery(this));
    });
  });
....

Idem par exemple pour les effets hover sur les images.

=> A utiliser dorénavant.
