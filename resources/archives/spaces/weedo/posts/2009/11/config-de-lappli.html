Config de l'appli

November 29 2009, 3:12 PM  by Karim Cassam Chena√Ø

On a souvent besoin de gérer des informations de configuration qui reviennent partout : N° de téléphone, nom du site, etc...

Pour gérer des données de configuration, voilà comment faire :
Les paramétrer dans app.yml

Exemple :=============
# default values
all:
.general:
websiteUrl: www.weedo.fr 
websiteName: weedo lucidea
=============

Et après, pour l'utilisation :

<?php echo sfConfig::get('app_websiteName') ?>


ça marche dans les templates.

Pour aller plus loin : lire : http://www.symfony-project.org/book/1_2/05-Configuring-Symfony
