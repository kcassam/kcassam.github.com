November 17 2009, 3:16 PM  by Karim Cassam Chenaï

Pour gérer les valeurs par défaut et la validation inline des champs, nous allons utiliser jquery, ainsi que les plugins jquery suivant :
http://plugins.jquery.com/project/defaultvalue
http://docs.jquery.com/Plugins/Validation

Tout est prêt dans le code pour l'utilisation, j'ai déjà chargé les plugins dans sf/apps/frontend/templates/layout.php, y'a plus qu'à les utiliser.
Merci de diffuser dans tout le site ces mécanismes.
