Weedo cherche un développeur web php zend et frontend

August 27 2012, 8:55 AM  by Karim Cassam Chenaï

En stage ou contrat d'apprentissage

À Paris, dés que possible, 9 mois minimum

Mission

Vous participerez notamment au développement d'un site important dans le domaine de l'immobilier de luxe

Dans le cadre de la mission, les techniques utilisées seront :

  • php5 (orientée objet)

  • frameworks PHP Zend

  • Mysql & ORM doctrine

  • javascript (ajax, jquery)

  • html5

  • css

Pour d'autres missions ponctuelles, vous serez peut-être amené(e) à utiliser

  • Symfony

  • CMS WordPress

Profil

Vous êtes passionné(e) par le développement web, rigoureux(se), courageux(se)

Qui sommes-nous

Weedo est une petite agence web

Nous développons des sites web pour des clients.

Notre priorité est de fournir un travail de haute qualité, en respectant les budgets et les délais.

Nous améliorons notre savoir-faire et nos pratiques en permanence.
